## Repo Intent

This repo packages a reusable Codex plugin for agent-driven delivery discipline and bundles a companion viewer under `apps/agent-log-viewer/`.

Use zoned separation because the repo now contains multiple ownership areas:
- plugin metadata and skills
- shipped templates
- transport scripts
- bundled viewer app
