# Docs Index

This repository uses the zoned Agent Ops Workbench layout.

## Reading Order

1. `docs/project/START_HERE.md`
2. `docs/project/BACKLOG.md`
3. the relevant file in `docs/specs/`
4. `agents/loop.md`

## Layout

- `docs/specs/` holds durable product and architecture specs
- `docs/project/` holds execution guidance, backlog, validation, workstreams, milestones, and decisions
- `agents/` holds planner/coder/evaluator loop guidance
